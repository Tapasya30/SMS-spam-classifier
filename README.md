# 🤖 AI SMS Spam Classifier

An intelligent Machine Learning web application that classifies SMS messages as **Spam** or **Not Spam (Ham)** in real time. A trained machine learning model to make accurate predictions through an interactive Streamlit interface.

---

## 🚀 Live Demo

🔗 **Live App:** *https://sms-spam-classifier-bik3fvpat6keq5pyesbuud.streamlit.app/*

---

## 📌 Features

* 📩 Detects whether an SMS message is Spam or Ham
* ⚡ Real-time prediction using Machine Learning
* 📊 Clean and interactive Streamlit interface
* 📱 Responsive and user-friendly design
* 🚀 Deployed online for public access

---

## 🧠 Machine Learning Pipeline

1. Data Cleaning
2. Text Preprocessing

   * Lowercasing
   * Tokenization
   * Removing punctuation
   * Removing stopwords
   * Stemming
3. Text Vectorization using TF-IDF
4. Model Training
5. Model Evaluation
6. Deployment using Streamlit
